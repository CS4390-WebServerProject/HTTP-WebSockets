What is PingPong?

PingPong is a minimal python web server with basic HEAD and GET support. It also includes gzip compression and caching to help get faster responses.

What are WebSockets?

WebSockets is a protocol for bidirectional communication between an existing HTTP client and a server. It allows soft real-time web applications to be developed and removes the the problem of polling HTTP servers for an update.
